# 📊 Sales Performance Dashboard

## 📌 Overview
This project analyzes monthly sales data using Python, Pandas, and Matplotlib.

## 🧠 Objective
To visualize business performance and extract meaningful insights from CSV data.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib

## 📂 Dataset
The dataset contains:
- Month
- Sales
- Profit
- Orders

## 📊 Visualizations
- Line Plot → Sales trend
- Bar Chart → Monthly profit
- Scatter Plot → Sales vs Profit
- Histogram → Sales distribution

## 🔍 Key Insights
- Sales increased from January to June
- June recorded the highest performance
- Higher sales correspond to higher profits
- Majority of sales values fall between 200–320

## 🚀 Conclusion
This project demonstrates how raw CSV data can be transformed into actionable insights using data visualization.
